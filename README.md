# VitrinY

VitrinY is an Etsy brand system for sellers who need stronger brand trust, better product presentation, and a professional destination for external traffic while keeping checkout on Etsy.

## Core positioning

VitrinY is not a cheaper Shopify replacement and not a generic theme marketplace.

It provides:

- Etsy-safe brand websites
- Product catalog and landing page structure
- Listing image and product visual systems
- Interactive demo directions by seller type
- Free audit flow for lead qualification
- Optional API-supported catalog assistance for suitable larger shops

## Current live system

Primary pages:

- `index.html` — main VitrinY brand system landing page
- `etsy.html` — VitrinY Commerce / Etsy brand website offer
- `studio.html` — VitrinY Studio / listing visuals and mockups
- `audit.html` — free audit lead form
- `portfolio.html` — focused showcase of core demo directions

Core demos:

- `demo-fashion-editorial.html` — Fashion Editorial Lookbook
- `demo-jewelry-gift.html` — Jewelry Gift Story

## API positioning

Starter brand pages do not require API setup.

Optional Etsy API-supported catalog assistance may be scoped for larger shops after access is approved, authorized, and technically connected.

Do not claim:

- automatic sync in every package
- always-updated catalog
- guaranteed API approval
- full Etsy automation without setup
